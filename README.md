# tankismus

The followup AI supported iteration building a top down panzer game in golang, using the ebiten engine and ecs pattern.

## structure


## features
- lighting
- collision detection using https://en.wikipedia.org/wiki/Hyperplane_separation_theorem
- location aware sound
- battlecards, powerups
- map generation
- gamemode: surivival / horde
